# vision_utils

GUI utils for Vision Dev(feat. React)

## Dependency

- `nvm`: 1.1.12
- `npm`: 10.9.0
- `package.json` for other dependency list

## 프로젝트 생성 방법

1. Vite 기반 프로젝트 생성

    ```
    npm create vite@latest
    ```

1. node 패키지 설치

    ```
    npm install
    ```

## 주요 npm 명령어

- 프로젝트 개발 서버 실행

    ```
    npm run dev
    ```

- 프로젝트 빌드

    ```
    npm run build
    ```

- 빌드 결과 테스트용 프리뷰

    ```
    npm run preview
    ```

## 기본 파일 목록

- `eslint.config.js`: ESLint 설정 파일
- `package.json`: 프로젝트 설정 파일
- `vite.config.js`: Vite 설정 파일
